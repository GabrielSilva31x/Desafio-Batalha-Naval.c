# ⚓ Batalha Naval em C

Projeto desenvolvido em **linguagem C** que simula um tabuleiro do jogo **Batalha Naval** utilizando matrizes.

O objetivo do projeto é praticar conceitos básicos de programação como:

* Matrizes
* Estruturas de repetição (`for`)
* Estruturas condicionais (`if / else`)
* Lógica de posicionamento em grade

---

# 📚 Tecnologias

* Linguagem **C**
* Biblioteca `stdio.h`

---

# 🧠 Estrutura do Projeto

O projeto possui **3 níveis de dificuldade**.

## 🟢 Nível Novato

Criação de um **tabuleiro 10x10** usando matriz.

* `0` representa água
* `3` representa navios

Inclui navios posicionados:

* Horizontalmente
* Verticalmente

---

## 🟡 Nível Aventureiro

Adiciona novos posicionamentos de navios:

* Horizontal
* Vertical
* Diagonal

Também inclui destaque visual das posições no terminal.

---

## 🔴 Nível Mestre

Implementa **habilidades especiais** usando matrizes **5x5** inseridas no tabuleiro principal.

Habilidades implementadas:

* Cone
* Cruz
* Octaedro

Essas formas são copiadas para a tabela principal **10x10** usando loops.

---

# 🎯 Objetivo

Treinar lógica de programação e manipulação de **matrizes bidimensionais** em C.
